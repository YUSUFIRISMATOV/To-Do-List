# Python To-Do List

A simple command-line To-Do List application built with Python. This project allows users to manage daily tasks efficiently with persistent JSON storage.

## Features

- ➕ Add new tasks
- 📋 View all tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 💾 Automatically save tasks to a JSON file
- 📅 Store the date and time each task was created
- 🖥️ Easy-to-use terminal interface

##  Technologies Used

- Python 3
- JSON
- os
- datetime

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/todo-list-python.git
```

2. Navigate to the project folder:

```bash
cd todo-list-python
```

3. Run the application:

```bash
python todo_listesi.py
```
